# Changelog

## 0.1.1 (2026-03-31)

- Standardize README to 3-badge format with emoji Support section
- Update CI actions to v5 for Node.js 24 compatibility
- Add GitHub issue templates, dependabot config, and PR template

## 0.1.0 (2026-03-21)

- Initial release
- Match file paths against glob patterns with `Glob.IsMatch`
- Filter collections with `Glob.Match` and `Glob.Filter`
- Support for `*`, `**`, `?`, character classes, brace expansion, and negation
- Configurable case sensitivity and path separator via `GlobOptions`
